# Gantt Chart Application

Приложение для управления задачами с визуализацией в виде Gantt-диаграммы.

## Структура проекта

- `backend/` - FastAPI backend
- `frontend/` - React frontend с Material-UI

## Установка и запуск

### Backend

```bash
cd backend
pip install -r requirements.txt
python main.py
```

Backend будет доступен на http://localhost:8001

### Frontend

```bash
cd frontend
npm install
npm start
```

Frontend будет доступен на http://localhost:3000

## Функционал

1. **Создание задачи:**
   - Название задачи
   - Тэг основной задачи
   - Год и месяц начала, год и месяц конца (поддерживается переход через границу года, например: декабрь 2025 - январь 2026)
   - Добавление подзадач (подзадачи автоматически получают длительность основной задачи)

2. **Gantt-диаграмма:**
   - Визуализация всех задач и подзадач
   - Отображение по месяцам с указанием года
   - Поддержка циклического календаря (переход через границу года)
   - Удаление задач прямо из диаграммы

3. **Экспорт:**
   - Экспорт Gantt-диаграммы в PDF
   - Экспорт задач в CSV (колонки: Задача/Подзадача, колонки с месяцами и годами)

## API Endpoints

- `GET /api/tasks` - Получить все задачи
- `GET /api/tasks/{task_id}` - Получить задачу по ID
- `POST /api/tasks` - Создать задачу
- `PUT /api/tasks/{task_id}` - Обновить задачу
- `DELETE /api/tasks/{task_id}` - Удалить задачу
- `GET /api/tasks/export/csv` - Экспорт в CSV
